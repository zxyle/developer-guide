## 变量

```shell
name=Tony

echo ${name}
echo "Hello ${name}!"

# 删除变量
unset name

echo ${name}
```



## 常量

```shell
name=Bob
readonly name

# 此处无法修改常量
name=Tom

echo ${name}
```



## 获取命令行参数

```shell
echo "脚本文件名: ${0}"
echo "第一个输入参数: ${1}"
echo "第二个输入参数: ${2}"

echo "输入参数个数:$#"
```
