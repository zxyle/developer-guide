## cargo
使用cargo创建项目
```bash
# 使用cargo创建项目
cargo new project_name

# 运行cargo项目
cargo run

# 编译cargo项目
cargo build

# 发布cargo项目
cargo build --release
```



修改crates源, 参考[USTC](https://mirrors.ustc.edu.cn/help/crates.io-index.html)或者[清华源](https://mirrors.tuna.tsinghua.edu.cn/help/crates.io-index.git/)
