## 转义符

| 符号 | 英文全称        | 作用   |
| ---- | --------------- | ------ |
| `\n` | newline         | 换行   |
| `\r` | carriage return | 回车   |
| `\\` | backslash       | 反斜杠 |
| `\t` | tabulation      | 制表符 |
| `\b` | backspace       | 退格符 |
| `\'` | single quote    | 单引号 |
| `\"` | double quote    | 双引号 |
| `\f` | formfeed        | 换页符 |
| `\s` | single space    | 空格   |
