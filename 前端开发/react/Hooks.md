

## ref

可以存储、查找组件内的标签或任意其他数据

```jsx
// 函数组件中使用
React.useRef()

// 类组件中使用
React.createRef()
```



## useState

函数组件 内部状态管理

```jsx
// count初始值为0，通过setCount来修改
const [count, setCount] = useState(0);
```
