## 数组遍历

```jsx
const songs = [
        { id: 1, name: "痴心绝对" },
        { id: 2, name: "像我这样的人" },
        { id: 3, name: "南山南" },
      ];

const list = (
    <ul>
        {songs.map((item, index) => (
            <li key={item.id}>{item.name}</li>
        ))}
    </ul>
);
```
