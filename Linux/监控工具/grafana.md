## 介绍
数据可视化平台, https://grafana.com/

安装包下载地址： https://mirrors.aliyun.com/grafana/



## CentOS安装

- `yum install grafana-7.4.3-1.x86_64.rpm`
- `systemctl enable grafana-server --now`



## Mac下安装

- `brew update`
- `brew install grafana`
- 启动: `brew services start grafana`
