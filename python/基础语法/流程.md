

## 条件判断

```python
if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

## match语法

python3.10新增语法，类似其他语言switch

```python
status_code = 900
match status_code:
    case 400:
        print("bad request")
    case 200:
        print("good")
    case _:
        print("Something else bad happened")
```
