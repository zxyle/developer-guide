## 示例

```python
with open('example.txt', encoding='utf-8') as f:
  f.read()
```



## 文件模式

|      |        |      |
| ---- | ------ | ---- |
| r    | 只读   |      |
| a    |        |      |
| w    | 只写   |      |
| b    | 二进制 |      |



### 读取大文件

```python
with open("example.txt", 'r') as fp:
    for line in fp:
        print(line, end="")
```
