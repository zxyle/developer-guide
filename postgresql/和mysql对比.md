|            | MySQL              | PSQL              |
| ---------- | ------------------ | ----------------- |
| 模型       | 单进程、多线程     | 多进程            |
| 默认用户名 | root               | postgres          |
| 默认端口   | 3306               | 5432              |
| 命令行登陆 | mysql -u 用户名 -p | psql -U 用户名 -W |
| 退出       | \q 或 exit         | \q                |
| 查看所有库 | show databases;    | \l                |
| 切换数据库 | use 库名;          | \c 库名           |
| 查看所有表 | show tables;       | \dt               |
| 查看表结构 | desc 表名;         | \d+ 表名          |
