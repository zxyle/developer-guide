## mac配置subl命令

```shell
# 第一步
sudo ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

# 第二步
alias subl="'/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl'"
```
