## 编写UDF

- 继承 `org.apache.hadoop.hive.ql.exec.UDF`
- 重写`evaluate`方法



```java
import org.apache.hadoop.hive.ql.exec.UDF;

public class AddPrefix extends UDF {
    /**
     * 这里我们实现将任一输入添加自定义前缀信息
     */
    public String evaluate(String str) {
        return "HIVE UDF Prefix:" + str;
    }
}
```
