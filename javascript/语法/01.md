## 介绍

一种解释型编程语言



## 两种打印输出

```javascript
alert("Hello World");

console.log("Hello World");
```



## 对话框

```javascript
const name = prompt("请输入姓名: ");
```





## 调用方式

- 外部文件
- 内嵌代码



## 注释

同java语言一样

```javascript
// 单行注释

/*
 多行
 注释
 （又叫块注释）
 */

/**
 * 文档注释
 */
```



## 变量

```javascript
// 声明一个变量
var age;

// 给变量赋值
age = 11;

// 声明并赋值
var name = 'zx'
```
