## 数字 number

包含整型和浮点型

```javascript
// 包含整数和小数
// 二进制 前面加0b
// 八进制 前面加0
// 十六进制 前面加0x

// 科学计数法
123e5

// 数字范围
Number.MAX_VALUE
Number.MIN_VALUE

// 无穷大，大于任何值
Infinity

// 无穷小，小于任何值
-Infinity

// Not a Number 代表一个非数值, 可以用 isNaN() 来判断
NaN
```





## 字符串 string

```javascript
// 可以是单引号 也可以是双引号

// 字符串长度
"hello".length

// 字符串拼接
"hello" + " world"

"hh" + 18
hh18
```



字符串方法

- charAt(索引号)
- replace(old, new);
- toUpperCase() 转换成大写
